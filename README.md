VPP Modular Middlebox (MMB 0.1)
===============================

## Vagrant

Install **Vagrant** on host machine.

## VirtualBox

Install **VirtualBox** on host machine.

## VPP-MB simulator

### Start the simulator

Still on host machine, go to *vagrant* directory (`vpp-mb/vagrant/`), and run:

    vagrant up
    vagrant ssh

### MMB Documentation

See *docs* directory.
