VPP Modular Middlebox
=====================

## Vagrant

Install **Vagrant** on your host machine.

## VirtualBox

Install **VirtualBox** on your host machine.

## VPP-MB simulator

### Start the simulator

In *vagrant* folder (`vpp-mb/vagrant/`), run:

    vagrant up
    vagrant ssh

### Documentation

See *docs* folder.
